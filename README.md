## Development

[![Greenkeeper badge](https://badges.greenkeeper.io/bayo-jsc/bilac.svg)](https://greenkeeper.io/)

+ Install dependencies:

```
go get -t -v ./...
yarn
```

+ Build at first time:

```
go build -i -v
```

+ Setup configs (example is default value):

```
BILAC_PORT=8080
```

+ Update build and start server:

```
./run.sh
```
